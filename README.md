# Food-Delivery-Analytics-Dashboard-Power-BI

Project Overview :

This project involves designing and developing an interactive Power BI dashboard to analyze food delivery data across multiple months (January to April). The dashboard provides meaningful business insights related to customer behavior, order trends, restaurant performance, and payment preferences using dynamic visualizations.

Objectives : 

1.Analyze monthly order trends to understand demand patterns

2.Identify top-performing restaurants based on order volume

3.Understand customer membership behavior (Gold vs Regular)

4.Evaluate popular food categories and payment methods

5.Create high-level KPIs for quick business insights

Dataset Description :

The dataset consists of multiple Excel files, each representing different months and supporting tables:

1.January_23.xlsx

2.February_23.xlsx

3.March_2023.xlsx

4.April_23.xlsx

5.Other_Data.xlsx (Customers, Food Items, Restaurants)

Key Tables Used : 

1.Orders Data

2.Customer Details

3.Food Details

4.Restaurant Details

Key Features & Insights : 

Total Orders: ~3,000

Food Items: 23

Restaurants: 6

UPI is the most preferred payment method

Main Course is the most ordered food category

Bukhara is the top-performing restaurant by order count

Highest orders observed in February

Dashboard Components : 

KPI Cards (Total Orders, Food Items, Restaurants)

Line Chart (Orders by Month)

Bar Charts (Food Type vs Membership, Payment Method)

Pie Charts (Food Category Distribution, Restaurant-wise Orders)

Slicers for dynamic filtering by Month
